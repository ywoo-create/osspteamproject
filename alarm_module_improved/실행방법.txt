정확도 보완 버전 실행 방법

1. Anaconda Prompt에서 환경 활성화
   conda activate test

2. 압축을 푼 폴더로 이동
   cd /d 폴더경로

3. 필요한 모듈 설치
   python -m pip install -r requirements_improved.txt

4. 실행
   python alarm_module_improved.py

개선 내용
- CNN(6_18).ipynb와 같은 노이즈 제거, 무음 제거, 피크 정규화 적용
- 3초 구간을 1.5초 간격으로 겹쳐 분석
- 최근 3회 예측 확률 평균 적용
- 소리가 작거나 신뢰도가 낮으면 분류 보류
- 버튼 없이 창이 열리면 자동으로 마이크 입력 시작
